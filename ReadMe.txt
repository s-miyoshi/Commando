Commandoについて

1.目的
  普段アルゴリズムを走らせてその出力にしか使っていないようなコマンドプロンプトの上でアクションゲームを動かしたらどうなるかと思い立ちこのゲームを制作した次第である.

2.概要
  Commandoはコマンドプロンプトで動くシューティングゲームである.
  コマンドプロンプトでの自由な文字描画にはWindows Console APIを使用している.
  描画関連はWinAPI.cppを参照されたい.

3.仕様
  本ゲームの仕様を以下に示す.
    エンドレス(いくら敵弾に当たっても終わらない)
    敵を倒すとスコア+1
    敵弾に当たるとスコア-10

4.操作方法
  初めに表示される"GAME OVER"画面にて"Space" Keyを押すとスタート,'Q' Keyを押すと終了.
  ゲームプレイ中の自機の操作は以下の通り
    'H' Key:左へ移動
    'L' Key:右へ移動
　　'A' Key:弾発射
